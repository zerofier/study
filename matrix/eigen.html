固有値と固有ベクトル

定義

`Av = lambda v`正方行列`A`に対しこの式を満足するベクトル`v`とスカラー`lambda`が存在すると
`v`を`A`の固有ベクトル`lambda`を固有値と呼ぶ

求めかた

定義により`(A - lambda I)v = 0`

`det(A - lambda I) = 0`式のなかの`lambda`をもとめて

例題：次の行列の固有値と固有ベクトルを求めよう

`[[3,2,4],[2,0,2],[4,2,3]]`

`det([[3 - lambda, 2, 4],[2, -lambda, 2], [4, 2, 3 - lambda]]) = 0`

余因子分解`(-1)^(1+1)(3 - lambda)det([[-lambda, 2], [2, 3 - lambda]]) + (-1)^(1+2)2det([[2,2],[4,3 - lambda]]) + (-1)^(1+3)4det([[2, -lambda],[4, 2]]) = 0`

`(3 - lambda)((-lambda)(3 - lambda) - 2*2) - 2(2(3 - lambda) - 2*4) + 4(2 * 2 - 4(-lambda)) = 0`

`(3 - lambda)(-lambda)(3 - lambda) - 4(3-lambda) - 4(3 - lambda) + 16 + 16 - 16(-lambda) = 0`

`(-lambda)(9 - 6 lambda + lambda^2) - 24 + 8 lambda + 32 + 16 lambda = 0`

`9(-lambda) - 6 lambda (-lambda) + lambda^2 (-lambda) - 24 + 8 lambda + 32 + 16 lambda = 0`

`- 9 lambda + 6 lambda^2 - lambda^3 - 24 + 8 lambda + 32 + 16 lambda = 0`

`- lambda^3 + 6 lambda^2 + 15 lambda + 8 = 0`

定数項が8なので`lambda`は`+-1,+-2,+-4,+-8`のなで解が一つ存在する。

`lambda = 8`の場合方程式を満足する。
