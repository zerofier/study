関数`y = ax + b`

誤差関数`e = 1/2 (y - t)^2`

即ち`e = 1/2 (ax + b - t)^2`

に置いて勾配は誤差関数の微分である

`del/(del a)(1/2(ax + b - t)^2) = (ax + b - t)(x)`

`(ax + b - t)(x) = 0`の場合誤差関数は最小値になる

`ax^2 + (b - t)x = 0`

`ax^2 = (t - b)x`

`a = (t - b) / x`の時が誤差最小になる

`del/(del b)(1/2(ax + b - t)^2) = (ax + b - t)`から同様

`ax + b - t = 0`

`b = t - ax`の時が誤差最小になる
